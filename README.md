# SQL User Management System

A full-stack web application to manage users using **Node.js, Express, MySQL, and EJS**.

It demonstrates CRUD operations, server-side rendering, and database integration.

---

##  Features

- View all users
- Add new users
- Edit user details
- Delete users with password verification
- Server-side rendering with EJS
- MySQL database integration

---

##  Tech Stack

- Node.js
- Express.js
- MySQL
- EJS
- Method Override
- HTML/CSS
